Long-Poll Sequence
1 App (foreground) starts startControlSupervisor()
2 Coroutine sends __fetch_push__ over QUIC
3 Server: queue empty → push_notify.notified() (blocks up to 60s)
4 External push → queue.push() → notify_one() → server wakes
5 Server pops message, streams to app, returns
6 App receives payload → applyPushPayload() → commitControlToDom()
7 Immediately starts next __fetch_push__ (no delay)
