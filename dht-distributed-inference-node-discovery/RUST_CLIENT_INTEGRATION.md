# Rust Client Integration Guide: Using Generated Node Configurations

## Note: DHT-Based Discovery

**Important**: In production, the cluster configuration will be **dynamically discovered through the Torrent DHT**, not from static JSON files. Nodes announce themselves to the DHT, and clients query the DHT to find available inference nodes. See `RUST_CLIENT_DHT_DISCOVERY.md` for DHT implementation details.

The JSON configurations generated by `generate_node_configs_fixed.ps1` can still be used for:
- **Bootstrap/Initial Discovery**: Seed nodes to start DHT queries
- **Development/Testing**: Skip DHT for local testing
- **Documentation**: Reference for expected cluster structure
- **Node Startup**: Nodes load their own config to know their role

---

This guide explains how to integrate the generated node configurations into your Rust client source code (for bootstrap/development use).

## Rust Client Implementation

```rust
use std::collections::HashMap;
use std::sync::Arc;
use anyhow::{Context, Result};
use quinn::{Endpoint, ClientConfig};
use rustls::{ClientConfig as RustlsClientConfig, RootCertStore};
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone)]
pub struct DistributedInferenceClient {
    config: ClusterConfig,
    entry_node: Option<NodeConfig>,
    exit_node: Option<NodeConfig>,
    endpoint: Arc<Endpoint>,
}

impl DistributedInferenceClient {
    /// Load cluster configuration from file
    pub fn new<P: AsRef<Path>>(config_path: P) -> Result<Self> {
        let content = fs::read_to_string(config_path.as_ref())
            .with_context(|| format!("Failed to read config from {:?}", config_path.as_ref()))?;
        
        let config: ClusterConfig = serde_json::from_str(&content)
            .context("Failed to parse cluster configuration JSON")?;
        
        let entry_node = config.nodes.iter()
            .find(|n| n.has_embeddings)
            .cloned();
        
        let exit_node = config.nodes.iter()
            .rev()
            .find(|n| n.has_output)
            .cloned();
        
        // Create QUIC endpoint
        let client_config = Self::create_client_config()?;
        let endpoint = Endpoint::client("0.0.0.0:0".parse().unwrap())?;
        
        Ok(Self {
            config,
            entry_node,
            exit_node,
            endpoint: Arc::new(endpoint),
        })
    }
    
    /// Create QUIC client configuration
    fn create_client_config() -> Result<ClientConfig> {
        let mut roots = RootCertStore::empty();
        // For development: accept self-signed certificates
        // In production, load proper CA certificates
        
        let rustls_config = RustlsClientConfig::builder()
            .with_safe_defaults()
            .with_root_certificates(roots)
            .with_no_client_auth();
        
        Ok(ClientConfig::new(Arc::new(rustls_config)))
    }
    
    /// Get the entry node (first node with embeddings)
    pub fn entry_node(&self) -> Option<&NodeConfig> {
        self.entry_node.as_ref()
    }
    
    /// Get the exit node (last node with output)
    pub fn exit_node(&self) -> Option<&NodeConfig> {
        self.exit_node.as_ref()
    }
    
    /// Get node by ID
    pub fn get_node(&self, node_id: u32) -> Option<&NodeConfig> {
        self.config.nodes.iter()
            .find(|n| n.node_id == node_id)
    }
    
    /// Get all nodes in pipeline order
    pub fn nodes(&self) -> &[NodeConfig] {
        &self.config.nodes
    }
    
    /// Process a query through the distributed pipeline
    pub async fn process_query(
        &self,
        query: &str,
        context: Option<&[Message]>,
        params: InferenceParams,
    ) -> Result<InferenceResponse> {
        let entry = self.entry_node()
            .ok_or_else(|| anyhow::anyhow!("No entry node found in configuration"))?;
        
        // Build QUIC connection address
        let addr = format!("{}:{}", entry.server_host, entry.server_port)
            .parse()
            .context("Invalid server address")?;
        
        // Connect to entry node
        let connection = self.endpoint
            .connect(addr, "localhost")? // Server name for TLS
            .await
            .context("Failed to establish QUIC connection")?;
        
        // Prepare request
        let request = InferenceRequest {
            query: query.to_string(),
            context: context.map(|c| c.to_vec()),
            parameters: params,
            pipeline_mode: true,
            target_node: self.exit_node.as_ref().map(|n| n.node_name.clone()),
        };
        
        // Send request
        let (mut send, mut recv) = connection
            .open_bi()
            .await
            .context("Failed to open bidirectional stream")?;
        
        let request_json = serde_json::to_vec(&request)?;
        send.write_all(&request_json).await?;
        send.finish().await?;
        
        // Read response
        let mut response_data = Vec::new();
        recv.read_to_end(&mut response_data).await?;
        
        let response: InferenceResponse = serde_json::from_slice(&response_data)
            .context("Failed to parse response")?;
        
        Ok(response)
    }
    
    /// Process query through pipeline manually (node-by-node)
    pub async fn process_manually(
        &self,
        query: &str,
        context: Option<&[Message]>,
        params: InferenceParams,
    ) -> Result<InferenceResponse> {
        let mut nodes = self.config.nodes.clone();
        nodes.sort_by_key(|n| n.node_id);
        
        let mut current_query = query.to_string();
        let mut current_activations: Option<Vec<u8>> = None;
        
        for (step, node) in nodes.iter().enumerate() {
            // Connect to node
            let addr = format!("{}:{}", node.server_host, node.server_port)
                .parse()
                .context("Invalid server address")?;
            
            let connection = self.endpoint
                .connect(addr, "localhost")?
                .await
                .context("Failed to connect to node")?;
            
            // Prepare pipeline request
            let request = PipelineRequest {
                query: current_query.clone(),
                context: context.map(|c| c.to_vec()),
                parameters: params.clone(),
                pipeline_step: step as u32,
                total_nodes: nodes.len() as u32,
                activations: current_activations.clone(),
                previous_shard_id: if step > 0 { Some(nodes[step - 1].shard_id) } else { None },
            };
            
            // Send request
            let (mut send, mut recv) = connection
                .open_bi()
                .await
                .context("Failed to open stream")?;
            
            let request_json = serde_json::to_vec(&request)?;
            send.write_all(&request_json).await?;
            send.finish().await?;
            
            // Read response
            let mut response_data = Vec::new();
            recv.read_to_end(&mut response_data).await?;
            
            let response: PipelineResponse = serde_json::from_slice(&response_data)
                .context("Failed to parse response")?;
            
            // Update for next node
            current_query = response.query.clone();
            current_activations = response.activations.clone();
            
            // If this is the exit node, return final result
            if node.has_output {
                return Ok(InferenceResponse {
                    answer: response.answer,
                    metadata: response.metadata,
                });
            }
        }
        
        Err(anyhow::anyhow!("Pipeline processing incomplete"))
    }
    
    /// Get cluster information
    pub fn cluster_info(&self) -> ClusterInfo {
        ClusterInfo {
            cluster_name: self.config.cluster_name.clone(),
            num_nodes: self.config.num_nodes,
            model_info: self.config.model_info.clone(),
            entry_node: self.entry_node.as_ref().map(|n| n.node_name.clone()),
            exit_node: self.exit_node.as_ref().map(|n| n.node_name.clone()),
        }
    }
}

// Request/Response structures
#[derive(Debug, Serialize, Deserialize)]
pub struct InferenceRequest {
    pub query: String,
    pub context: Option<Vec<Message>>,
    pub parameters: InferenceParams,
    pub pipeline_mode: bool,
    pub target_node: Option<String>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct InferenceParams {
    pub temperature: f32,
    pub max_tokens: u32,
    pub top_p: f32,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct Message {
    pub role: String,
    pub content: String,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct InferenceResponse {
    pub answer: String,
    pub metadata: ResponseMetadata,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct ResponseMetadata {
    pub model: Option<String>,
    pub shard_id: Option<u32>,
    pub layers_processed: Option<String>,
    pub tokens_generated: Option<u32>,
    pub latency_ms: Option<f64>,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct PipelineRequest {
    pub query: String,
    pub context: Option<Vec<Message>>,
    pub parameters: InferenceParams,
    pub pipeline_step: u32,
    pub total_nodes: u32,
    pub activations: Option<Vec<u8>>,
    pub previous_shard_id: Option<u32>,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct PipelineResponse {
    pub query: String,
    pub answer: String,
    pub activations: Option<Vec<u8>>,
    pub metadata: ResponseMetadata,
}

#[derive(Debug, Clone)]
pub struct ClusterInfo {
    pub cluster_name: String,
    pub num_nodes: u32,
    pub model_info: ModelInfo,
    pub entry_node: Option<String>,
    pub exit_node: Option<String>,
}
```

## Usage Example

```rust
use anyhow::Result;
use tokio;

#[tokio::main]
async fn main() -> Result<()> {
    // Load cluster configuration
    let client = DistributedInferenceClient::new("./node_configs/cluster_config.json")?;
    
    // Display cluster information
    let info = client.cluster_info();
    println!("Cluster: {}", info.cluster_name);
    println!("Nodes: {}", info.num_nodes);
    println!("Entry Node: {:?}", info.entry_node);
    println!("Exit Node: {:?}", info.exit_node);
    
    // Process a query
    let params = InferenceParams {
        temperature: 0.7,
        max_tokens: 512,
        top_p: 0.9,
    };
    
    let context = vec![
        Message {
            role: "system".to_string(),
            content: "You are a helpful assistant.".to_string(),
        },
    ];
    
    let response = client
        .process_query(
            "What is the capital of France?",
            Some(&context),
            params,
        )
        .await?;
    
    println!("Answer: {}", response.answer);
    println!("Metadata: {:?}", response.metadata);
    
    Ok(())
}
```

```rust
/// Load configuration for a specific node
pub fn load_node_config<P: AsRef<Path>>(
    node_id: u32,
    config_dir: P,
) -> Result<NodeConfig> {
    let config_file = config_dir.as_ref().join(format!("node-{:02}.json", node_id));

        .with_context(|| format!("Failed to read node config from {:?}", config_file))?;
    
    let config: NodeConfig = serde_json::from_str(&content)
        .context("Failed to parse node configuration")?;
    
    Ok(config)
}
```

## Connection Pooling

For better performance, reuse connections:

```rust
use std::collections::HashMap;
use std::sync::Arc;
use tokio::sync::RwLock;

pub struct ConnectionPool {
    connections: Arc<RwLock<HashMap<String, Arc<Endpoint>>>>,
}

impl ConnectionPool {
    pub fn new() -> Self {
        Self {
            connections: Arc::new(RwLock::new(HashMap::new())),
        }
    }
    
    pub async fn get_connection(
        &self,
        host: &str,
        port: u16,
    ) -> Result<Arc<Endpoint>> {
        let key = format!("{}:{}", host, port);
        
        // Check if connection exists
        {
            let conns = self.connections.read().await;
            if let Some(endpoint) = conns.get(&key) {
                return Ok(endpoint.clone());
            }
        }
        
        // Create new connection
        let endpoint = Arc::new(Endpoint::client("0.0.0.0:0".parse().unwrap())?);
        
        {
            let mut conns = self.connections.write().await;
            conns.insert(key, endpoint.clone());
        }
        
        Ok(endpoint)
    }
}
```

## Complete Example with Error Handling

```rust
#[tokio::main]
async fn main() -> Result<()> {
    // Initialize client
    let client = match DistributedInferenceClient::new("./node_configs/cluster_config.json") {
        Ok(client) => client,
        Err(e) => {
            eprintln!("Failed to load configuration: {}", e);
            return Err(e);
        }
    };
    
    // Verify entry node exists
    let entry = client.entry_node()
        .ok_or_else(|| anyhow::anyhow!("No entry node configured"))?;
    
    println!("Connecting to entry node: {}:{}", entry.server_host, entry.server_port);
    
    // Process query with timeout
    let params = InferenceParams {
        temperature: 0.7,
        max_tokens: 512,
        top_p: 0.9,
    };
    
    let response = tokio::time::timeout(
        std::time::Duration::from_secs(120),
        client.process_query(
            "Explain quantum computing in simple terms",
            None,
            params,
        )
    ).await??;
    
    println!("Response: {}", response.answer);
    println!("Processed by shard: {:?}", response.metadata.shard_id);
    
    Ok(())
}
```

## Integration into Your Rust Project

1. **Add the configuration loading module** to your project
2. **Copy the structures** from this guide into your codebase
3. **Use the client** in your application code:

```rust
// In your main application
mod inference_client;
use inference_client::DistributedInferenceClient;

// Initialize once at startup
let client = DistributedInferenceClient::new("node_configs/cluster_config.json")?;

// Use throughout your application
let response = client.process_query("Your question", None, params).await?;
```

The generated configurations provide all the information needed to connect to and communicate with your distributed inference nodes.
